# Contributing

Read the organization-wide contribution guide in the LizardByte developer docs:
[https://docs.lizardbyte.dev/latest/developers/contributing.html](https://docs.lizardbyte.dev/latest/developers/contributing.html)

## Project Direction

This repository is now focused on a minimal remote desktop streaming server. Contributions should align with that
scope:

- desktop capture
- video/audio streaming
- Moonlight compatibility
- keyboard and mouse input
- terminal-based administration
- packaging and deployment for headless or automated installs

## Avoid Expanding Back Into the Removed Surface

The following are no longer part of the intended product direction:

- browser-based configuration UI
- app launching and app catalogs
- featured apps
- controller or gamepad setup flows
- gaming-specific tuning surfaces
